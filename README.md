pixelwars
=========

Projet BTS, jeu 2d

[Source documentation here.](https://xabufr.github.io/pixelwars/doxygen/index.html)

#Dependencies

* SFML 2.0
* Box2D
* libNoise
* tinyXML


#Compilation
``` bash
cmake .
make -jX # where X is your number of processor cores
```

#Run
``` bash
cd build
./pixelwars
```
